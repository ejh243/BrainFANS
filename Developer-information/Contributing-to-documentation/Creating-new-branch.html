Step 2: Creating a new branch

Cloning the repository

If you do not already have the BrainFANS repository on your local machine, please refer to the existing documentation on this before continuing.

Start by creating a new branch off of the master branch by using the following git commands in the terminal:

# Change your directory to the position of your local BrainFANS repository
cd /path/to/BrainFANS

# Fetch the latest changes to the repository
git fetch

# Create a new branch off of web-development and move to it
git checkout -b your-branch-name origin/master

Now you are on the new branch you can start making commits.
